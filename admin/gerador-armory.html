Gerador de Markdown de Resultados

Crie arquivos .md para Armory, Liga e Torneio com os campos usados pelo site. O título, resumo, record, imagem alternativa e caminho sugerido são preenchidos automaticamente.

Dados do evento

Destino automático: Armory vai para _comunidade/Armory, Liga para _comunidade/Liga e Torneio para _comunidade/Torneio.
Record automático: vitórias - (rodadas menos vitórias). Exemplo: 4 rodadas e 3 vitórias gera 3-1.
Campos reaproveitados: jogadores, locais e heróis/decks vêm dos resultados já cadastrados no site.

Resultados dos jogadores

Use o campo de jogador para selecionar alguém que já participou. Se for jogador novo, é só digitar o nome. O campo de herói/deck muda conforme jogo e formato.
