2. Spikes
- Página configurada para mostrar apenas os spikes das últimas 24 horas.
- O limite visual está preparado para até 9 cartas por jogo.
- Continua usando assets/data/spikes.json.
- Para atualizar dados reais, configure JUSTTCG_API_KEY nos Secrets do GitHub e rode o workflow manualmente.
- O workflow automático está agendado para 05:00 no horário de Brasília.

3. Comunidade / Home
- A Home mantém o bloco "Últimas imagens da comunidade" como estava.
- O bloco "Últimas postagens" agora tem altura mínima nos cards.
- Posts de Instagram no bloco "Últimas postagens" ficam compactos para não quebrar o layout.
