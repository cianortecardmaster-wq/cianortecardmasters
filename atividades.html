Atividades

Área para reunir caça-palavras, palavras cruzadas, quiz, picross e outras atividades do Cianorte Card Masters. O fluxo é igual ao Blog, Decks e Comunidade: criou um arquivo .md dentro de _atividades/, publicou no front matter e ele entra na listagem automaticamente.

Sem mexer em planilha YAML

Publique criando apenas arquivos .md

Para adicionar uma nova atividade, crie um arquivo em uma subpasta de _atividades/, preencha os campos do front matter e cole abaixo o conteúdo da atividade. Pode ser Markdown, HTML, CSS e JavaScript puro.

Os filtros são montados pelo JavaScript lendo os próprios cards publicados. Quando você usar um novo valor em activity_type ou game, a página passa a mostrar esse novo filtro automaticamente.

Exemplo de pasta _atividades/Flesh-and-Blood/

Use nomes claros, como caca-palavras-omens.md, picross-dash-io.md ou quiz-pokemon-cianorte.md. Se quiser esconder um rascunho, use published: false.
