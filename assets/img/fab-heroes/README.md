# Ícones de heróis FAB

A página Comunidade já busca imagem dos heróis pela API GoAgain quando o Armory é renderizado.

Se quiser usar os ícones locais enviados em pacote próprio, coloque aqui arquivos `.webp` ou `.png` com nomes simplificados, por exemplo:

- `arakni-huntsman.webp`
- `arakni-marionette.webp`
- `pleiades-superstar.webp`
- `ira-scarlet-revenger.webp`
- `hala-bladesaint-of-the-vow.webp`

Também é possível informar uma imagem manual em cada resultado do Markdown usando `hero_icon: "/assets/img/fab-heroes/nome-do-arquivo.webp"`.
